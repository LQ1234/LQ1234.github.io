Particlesys (func, generator, boxsize):

func: Function to be ran on every particle. Ran like function(particle, world) where particle is this particle and world is a clone of this Particlesys.
generator: Function that generates the particles.
boxsize: size of boxes that particles are sorted in.

clone ():

Clones this Particlesys.
returns: clone.

add (particle):

particle: Particle to add.
returns: nothing.

forEach (func):

func: Function to run on each particle.
returns: nothing.

remove (particle):

particle: Particle to remove.
returns: particle to remove, or null if particle is not found.

reset ():

Removes all particles and reruns generator.
returns: nothing.

getWithinRange (pops, no):

pops: Particle that is center of the particles returned
no: distance of particles around center.
Note: searches in a box, and returns whole grid boxes
returns: list of particles.

tick ():

Runs the function on every particle, adds the velocitys to coordinates, and moves particle to correct box.
returns: nothing.


Particle (size, x, y, xv, yv):

size: size of particle.
x, y: Coordinates of particle.
xv, yv: Velocity of particle.


generators object

Object exposed that is used in Particlesys constructor.

generators.grid (width, height, sep) :

width, height: Size of grid of particles.
sep: Distance between particles in grid.

generators.grid (width, height, sep) :

Creates grid of particles.
width, height: Size of grid of particles.
sep: Distance between particles in grid.

generators.random (width, height, amount) :

Creates particles in random position.
width, height: Boundaries of which particles are generated.
amount: Amount of particles to generate.

generators.randomDir (width, height, amount, randomness) :

Does the same as generators.random, but the velocities of particles are random.
randomness: The speed of the particle.

generators.allOut (force, amount, randomness) :

All of the particles spawn at origin
force: The speed of the particles.
amount: Amount of particles.
randomness: How exact the force is.
